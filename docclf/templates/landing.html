Mortgage Document Classifier

This app uses a trained Machine Learning model to classify Mortgage Documents.

Documents must be submitted as a .txt or .csv file where each line of the file contains the words of a single document.

If no labels are provided, the ouput will be only the predicted class of document. If labels are provided, the predicted and actual classes will be outputted, along with the average accuracy for that batch of files.

The model currently supports the following classes:

  • DELETION OF INTEREST
  • RETURNED CHECK
  • BILL
  • POLICY CHANGE
  • CANCELLATION NOTICE
  • DECLARATION
  • CHANGE ENDORSEMENT
  • NON-RENEWAL NOTICE
  • BINDER
  • REINSTATEMENT NOTICE
  • EXPIRATION NOTICE
  • INTENT TO CANCEL NOTICE
  • APPLICATION
  • BILL BINDER
